## 설계 메모

- `pyproject.toml`에는 이미 `pytest`, `pytest-asyncio`, `testpaths = ["tests"]`, `asyncio_mode = "auto"`가 있어 추가 설정 없이 전환 가능했다.
- production/runtime 코드는 건드리지 않고, 검증 자산만 pytest 스타일로 이동했다.
- 기존 smoke scripts는 참고 자산으로 유지하고, 새 기준 실행 경로는 `uv run pytest tests/`로 정리했다.
