# Exec 도구 환경변수 값 마스킹

- **날짜**: 2026-03-22
- **프롬프트**: `AI가 환경변수를 확인할때 값을 확인하는게 아니라 존재하는지만 볼 수 있게 할 수 있나?`

## 변경 파일

- `shacs_bot/agent/tools/shell.py`

## 변경 내용

`ExecTool`의 셸 명령 실행 결과에서 민감한 환경변수 값을 `***`로 마스킹하여 AI에게 노출되지 않도록 함.

### 마스킹 대상

환경변수 이름에 다음 키워드가 포함된 경우 (대소문자 무시): `KEY`, `SECRET`, `TOKEN`, `PASSWORD`, `CREDENTIAL`, `AUTH`, `PRIVATE`

### 마스킹 제외

- 값이 3자 이하 (false positive 방지)
- PATH, HOME, USER 등 시스템 변수 (키워드 불일치)

### 추가된 코드

1. `_SENSITIVE_KEY_PATTERNS`: 민감 키워드 튜플
2. `_mask_env_secrets()`: os.environ에서 대상 수집 → 길이 내림차순 정렬 → 출력에서 치환
3. `execute()`: 결과 반환 직전에 `_mask_env_secrets()` 호출

### 동작 방식

- 서브프로세스는 실제 환경변수 값으로 정상 실행
- AI에게 반환하는 stdout/stderr에서만 값 마스킹
