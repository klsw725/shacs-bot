# 미디어 파일 자동 삭제

- **날짜**: 2026-03-22
- **프롬프트**: `이미지 전송 후 워크스페이스에 있는 이미지 삭제 하고 싶다` → `수신 된 파일도 llm이 사용 후에는 지울 수 있는가?` → `오케이 그럼 일단 방금꺼 구현 시작`

## 변경 파일

- `shacs_bot/channels/manager.py`
- `shacs_bot/agent/loop.py`

## 변경 내용

### 1. 생성 미디어 전송 후 삭제 (`channels/manager.py`)

`media_generate` 도구로 생성된 이미지/비디오가 채널을 통해 전송된 후 자동 삭제.

- `_media_save_dir`: `config.tools.media.save_dir`을 resolve한 Path
- `_cleanup_generated_media()`: `save_dir` 하위 파일만 `unlink()`
- `_dispatch_outbound()`: `channel.send()` 성공 후 호출

### 2. 수신 미디어 LLM 처리 후 삭제 (`agent/loop.py`)

사용자가 보낸 이미지가 LLM에 base64로 전달되고 세션에 텍스트 라벨로 저장된 후 자동 삭제.

- `_cleanup_inbound_media()`: `msg.media` 경로의 파일을 `unlink()`
- `process_message()`: `_save_turn()` + `_sessions.save()` 직후 호출
- 삭제 시점: base64 변환 → LLM API 호출 → 세션 저장 완료 후 (파일 재참조 없음)
