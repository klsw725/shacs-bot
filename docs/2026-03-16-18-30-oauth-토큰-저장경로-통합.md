# OAuth 토큰 저장 경로 통합

## 프롬프트

> nanobot 설정을 가져온거야 codex 저 로그인 기록을 config.json에 남겨서 저장하는 방법이 있어?

## 변경 요약

`oauth-cli-kit`이 OS별 다른 경로(`platformdirs`)에 저장하던 OAuth 토큰을 `~/.shacs-bot/auth/codex.json`으로 통합.

Docker에서 `~/.shacs-bot/` 볼륨 마운트만으로 설정 + 인증 모두 유지 가능.

## 변경 파일

| 파일 | 내용 |
|---|---|
| `shacs_bot/providers/openai_codex.py` | `codex_token_storage()` 헬퍼 추가, `get_codex_token()` 호출 시 storage 전달, 레거시 마이그레이션 로직 |
| `shacs_bot/cli/commands.py` | `_login_openai_codex()`에서 `get_token()`과 `login_oauth_interactive()`에 storage 전달 |
| `docs/prds/oauth-token-storage-consolidation.md` | PRD 작성 |

## Before → After

```
# Before
~/Library/Application Support/oauth-cli-kit/auth/codex.json  (macOS)
~/.local/share/oauth-cli-kit/auth/codex.json                 (Linux/Docker)

# After
~/.shacs-bot/auth/codex.json  (모든 OS 통일)
```

## 마이그레이션

기존 경로에 토큰이 있고 새 경로에 없는 경우 자동 이전됨 (`_migrate_legacy_token()`).
