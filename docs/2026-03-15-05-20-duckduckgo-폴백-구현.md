# DuckDuckGo 폴백 웹 검색 구현

> **Prompt**: nanobot 원본에 있는 DuckDuckGo 폴백 기능을 shacs-bot에 포팅

## 변경 파일

| 파일 | 변경 |
|---|---|
| `pyproject.toml` | `ddgs>=9.5.5,<10.0.0` 의존성 추가 |
| `shacs_bot/agent/tools/web.py` | `_search_duckduckgo()`, `_format_results()` 메서드 추가. `execute()`에서 BRAVE_API_KEY 미설정 시 DuckDuckGo 폴백. |
| `docs/prds/duckduckgo-fallback.md` | PRD 작성 + M1/M2 완료 표시 |

## 핵심 변경 내용

### Before
- `BRAVE_API_KEY` 없으면 에러 메시지 반환, 웹 검색 불가

### After
- `BRAVE_API_KEY` 없으면 `logger.warning` 출력 후 DuckDuckGo 폴백
- `ddgs.text()` → `asyncio.to_thread()`로 비동기 래핑
- `_format_results()` 공통 포맷 메서드로 Brave/DuckDuckGo 결과 형식 통일
- BRAVE_API_KEY 있으면 기존과 동일하게 Brave API 사용 (하위 호환)

## 테스트 결과

```
WARNING  | BRAVE_API_KEY not set, falling back to DuckDuckGo
Tool call: web_search({"query": "AI agent framework 2025", "count": 5})
→ 5개 검색 결과 정상 반환
```
